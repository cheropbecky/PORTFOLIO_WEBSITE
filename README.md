# My Portfolio

A modern, responsive, and interactive portfolio website built with React and Tailwind CSS. This project showcases my skills, projects, and services as a Software Engineering student and Frontend Developer.

## Features

- **Modern UI/UX**: Clean, dark-themed design with glassmorphism effects and smooth animations.
- **Responsive Design**: Fully optimized for all devices, from mobile phones to large desktops.
- **Interactive Elements**:
    - Animated section headings.
    - Hover effects on buttons and cards.
    - Smooth scrolling navigation.
- **Dynamic Content**:
    - **Hero Section**: Engaging introduction with animated text and profile visual.
    - **About**: Overview of skills and background.
    - **Services**: Details of services offered (Web Development, Redesign, etc.).
    - **Projects**: Showcase of featured work with links to demos and GitHub repositories.
    - **Contact**: Functional contact form (via `mailto`) and social media links.
- **CV Download**: Direct download link for my resume.
- **Theme Support**: Light and Dark mode toggle.

## Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/cheropbecky/portfolio.git
    cd portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```
